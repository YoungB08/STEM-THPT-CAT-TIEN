KNTech Physics 12 Simulator
==========================

Phần mềm mô phỏng Vật Lí 12 (Nhiệt học, Khí lí tưởng, Từ trường, Hạt nhân)
Thương hiệu: KNTech

Công nghệ:
- Python
- PyQt5 (GUI gradient hiện đại)
- VPython (3D simulation – sẽ tích hợp)

Cách chạy:
1. pip install pyqt5 vpython numpy
2. python main.py
